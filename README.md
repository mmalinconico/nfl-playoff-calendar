# NFL Playoff Calendar

An automatically updating iCalendar (.ics) subscription for the NFL postseason.

## Included Events

- Wild Card Round
- Divisional Round
- AFC Championship
- NFC Championship
- Super Bowl

As playoff teams, kickoff times, venues, cities, and television networks are announced, the calendar updates automatically.

## Features

- Automatically retrieves postseason schedule information from ESPN.
- Checks monthly during the offseason.
- Updates every 4 hours during the NFL playoff season (December through February).
- Generates a standards-compliant iCalendar (.ics) feed.
- Hosted with GitHub Pages.
- Works with Apple Calendar, Google Calendar, Outlook, and other calendar apps that support subscriptions.
